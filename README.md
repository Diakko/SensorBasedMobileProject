# SensorBasedMobileProject
A school project for the course "Sensor Based Mobile Applications" (more info here: 
https://opinto-opas.metropolia.fi/en/realization/TX00CK66-3009)

### INGREDIENTS

- Retrofit
- Room DB
- LiveData
- ViewModel
- Coroutines
- Navigation
- RecyclerView
- Fragments
- API connections: Fineli
- GPS
- Camera
